# XChange

### Description
XChange is a java console app that I built that consumes the Exchange Rate API to perform real-time currency conversions.

## Functions
- Console user menu:
  1. Welcome message
  2. User menu
     
- `Currency conversion options`:
  1. USD => CRC
  2. CRC => USD
  3. USD => MXN
  4. MXN => USD
  5. USD => EUR
  6. EUR => USD
